# Wardrobe Sliding

## When to use
- User asks for cua lua, sliding closet, or small bedroom wardrobe.

## Recipe
1. Preview current model.
2. Add `sliding-2door` for 120-240 cm widths.
3. Add `sliding-3door` for spans above 240 cm.
4. Add optional `open-bookshelf` at one side for display.

## Common pitfalls
- Sliding door modules should keep full height and shallow handles.
- Do not add swing-door clearance in front.
