# Child Bedroom

## When to use
- User asks for phong ngu tre em or a child bedroom storage wall.

## Recipe
1. Use `sliding-2door` for safe compact wardrobe storage.
2. Use `l-desk-return` or raw desk boxes for a study area.
3. Add `open-bookshelf` for books and toys.
4. Use lighter palettes such as `laminate-white` unless user requests wood.

## Common pitfalls
- Avoid tall handles at child head height.
- Keep circulation and desk seating space open.
