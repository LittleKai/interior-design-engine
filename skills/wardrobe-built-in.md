# Wardrobe Built-in

## When to use
- User asks for tu am tuong or full-height wardrobe.

## Recipe
1. Use one `east` run.
2. Add `sliding-2door` or `sliding-3door` for main closet spans.
3. Add `upper-2door` above low areas when needed.
4. Use `open-bookshelf` for display shelves.
5. Commit with a short Vietnamese explanation.

## Common pitfalls
- Keep depth 55-65 cm for hanging clothes.
- Use sliding templates for narrow room circulation.
